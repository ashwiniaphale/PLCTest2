# PLC Test 2

### Regular Expression: 
* Addition: +
* Subtraction: - 
* Multiplication: * 
* Division: /
* Module: %
* Less than: <
* Greater than: >
* Less than Equal To: #
* Greater than Equal To: $
* Equal To: ~
* Not Equal To: !
* Assignment: =
* (There must also be a way to break precedence, this is usually done with the use of parenthesis): ([a-zA-Z0-9_/+-*% ]*)

### Variable Names:
[a-z_A-Z] [a-z_A-Z] [a-z_ A-Z] [a-z_ A-Z] [a-z_ A-Z] [a-z_ A-Z] [a-z_ A-Z]?[a-z_ A-Z]?
### Integer Literals:
1 byte: [0-9]+_b
2 byte: s_[0-9]+_s
4 byte: i_[0-9]+_i
8 byte: [0-9]+_l
